# Students API

A RESTful backend API for managing student records, built with Node.js and Express.js.

This project was developed as part of a structured 30-day Backend Developer roadmap (Days 13–14).

---

## 🛠️ Tech Stack

- **Runtime:** Node.js
- **Framework:** Express.js
- **Architecture:** MVC (Controllers / Routes)
- **Data Storage:** In-memory array (no database)

## 🚀 Getting Started

### Prerequisites

- Node.js installed
- npm installed

### Installation

```bash
git clone https://github.com/Mu1ie/students-api.git
cd students-api/day13-14/students-api
npm install
```

### Run the Server

```bash
node src/index.js
```

Server runs on: `http://localhost:3000`

## 📡 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/students` | Get all students |
| GET | `/students/:id` | Get a student by ID |
| POST | `/students` | Create a new student |
| PUT | `/students/:id` | Update a student |
| DELETE | `/students/:id` | Delete a student |

---

### Request & Response Examples

#### GET /students
```json
[
  { "id": 1, "name": "Ali Hassan", "age": 20, "grade": "A" },
  { "id": 2, "name": "Sara Ahmed", "age": 22, "grade": "B" }
]
```

#### POST /students
**Request Body:**
```json
{
  "name": "Layla Mohammed",
  "age": 23,
  "grade": "B"
}
```
**Response:** `201 Created`
```json
{ "message": "Student added Successfully" }
```

#### PUT /students/:id
**Request Body:**
```json
{
  "name": "Ali Hassan Updated",
  "age": 21,
  "grade": "A+"
}
```
**Response:** `200 OK`
```json
{ "message": "Student updated" }
```

#### DELETE /students/:id
**Response:** `200 OK`
```json
{ "message": "Student deleted" }
```

#### Error Response (Student Not Found)
```json
{ "message": "Student not found" }
```
Status: `404 Not Found`
